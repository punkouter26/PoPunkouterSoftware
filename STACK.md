# Technology Stack

- .NET 10 / C# 14 with nullable references and warnings treated as errors.
- ASP.NET Core host with Blazor WebAssembly interactive components.
- Radzen.Blazor for UI components and the repository's custom CSS design system.
- Azure SDKs for ARM, identity, monitoring, App Service, Resource Graph-adjacent inventory, and Table Storage.
- Serilog and OpenTelemetry with optional Azure Monitor export.
- xUnit, FluentAssertions, WebApplicationFactory, Azurite/Testcontainers, and Playwright.
- PowerShell 7 and Azure CLI for user-run Azure automation assets.
