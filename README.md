## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/PoPunkouterSoftware.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PoPunkouterSoftware
   ```
3. Install the necessary dependencies:
   ```bash
   dotnet restore
   ```
4. Build the project:
   ```bash
   dotnet build
   ```
5. Run the project:
   ```bash
   dotnet run
   ```
